# Navia-Browser
### Navegador web en Python con GTK y WebKit – modular, rápido y extensible

Navia Browser es un navegador web libre escrito en **Python** usando **GTK y WebKitGTK**. 
Es el **proyecto principal** del ecosistema Navia/Fennex, con un sistema de **add-ons internos**, soporte para pestañas, historial, marcadores, descargas y configuración avanzada.
